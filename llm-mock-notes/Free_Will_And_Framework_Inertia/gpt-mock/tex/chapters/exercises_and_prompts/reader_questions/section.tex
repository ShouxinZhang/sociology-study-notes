\section{Fast Answers for Common Action Problems}

This section is a rapid-entry layer for readers who need a usable answer before
they need a full theoretical tour. Each question starts from a common failure
pattern, gives an immediate operational answer, and translates the book's core
ideas into a short sequence of moves. The value of this design is pedagogical
compression: a reader can start with the current problem, stabilize one
transition, and then return to the earlier chapters for deeper explanation.
