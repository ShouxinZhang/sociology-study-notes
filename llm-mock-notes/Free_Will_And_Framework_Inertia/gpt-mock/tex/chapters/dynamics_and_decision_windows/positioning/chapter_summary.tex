\begin{summarybox}
This chapter explains why raw effort is a poor operating principle. Timing
dominates intensity because free will is not best modeled as a continuously
available force reservoir. It is better modeled as a scheduling problem:
intervene where barrier geometry is already favorable, begin before the window
closes, and treat repeated low-barrier opportunities as scarce productive
assets. Missed windows are expensive not because the agent lacks values, but
because the system has already restabilized around the old frame.
\end{summarybox}
