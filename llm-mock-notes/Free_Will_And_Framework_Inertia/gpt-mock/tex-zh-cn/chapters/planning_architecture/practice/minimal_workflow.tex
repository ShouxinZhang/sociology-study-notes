\section{章级最小工作流}

\begin{templatebox}
\textbf{最小转移计划}
\begin{enumerate}[nosep]
  \item 写出目标状态。
  \item 写出最可能出现的决策窗口。
  \item 写下第一个可见动作。
  \item 把所需物品提前放入未来环境。
  \item 定义这次会话的停止规则。
\end{enumerate}
\end{templatebox}

\begin{remark}
这个工作流是教学性的，而不是定理式的。它的正当性在于：每一行都对应上文形式化出的某个变量。目标状态固定目标，决策窗口固定时机，第一个动作降低含糊性，环境摆放提高准备，停止规则则限制会话内部重新设计的成本。
\end{remark}
