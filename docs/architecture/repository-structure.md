# Repository Structure

## Root Directory
- `notes/`: Main module containing categorized study notes and reflections.
  - `Methodology_Academic_Strategy/`: Notes related to learning methods, mathematics, and productivity strategies.
    - `Learning_Cognition.md`: Cognitive load, feedback, learning strategies.
    - `Mathematics_Academic.md`: Math studies, probability, formalization.
    - `Productivity_Strategy.md`: Work efficiency, habits, long-term strategy.
  - `Philosophy_Life_Reflection/`: Notes on personal philosophy, life narratives, and tools.
    - `Personal_Growth_Philosophy.md`: Inner growth, dreams, mindset.
    - `Life_Tools_Misc.md`: Daily life observations, specialized tools.
- `1.md`: Original notes file (deprecated/to be processed).
- `AGENTS.md`: Agent instructions.
- `LICENSE`: License file.

## Docs
